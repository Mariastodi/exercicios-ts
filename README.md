# Dio Bank - Exercícios TypeScript

Este repositório contém a resolução dos quatro desafios propostos no bootcamp de TypeScript.

---

## Desafio 01: Conceitos Básicos de TS
Projeto focado em POO (Programação Orientada a Objetos) e fundamentos.

**Principais Desafios:**
- [x] Implementar métodos de depósito e saque na `DioAccount`.
- [x] Implementar empréstimo na `CompanyAccount`.
- [x] Criar novo tipo de conta com bônus de depósito.
- [x] Encapsulamento de atributos (Private e Readonly).

---

## Desafio 02: React com TypeScript
Refatoração de componentes e criação de interface inicial.

**Principais Desafios:**
- [x] Criação de componentes de Header, Button e Card.
- [x] Refatoração do formulário de login.
- [x] Testes unitários para função de boas-vindas.

---

## Desafio 03: Context API e LocalStorage
Implementação de sistema de login e persistência de dados.

**Principais Desafios:**
- [x] Validação de senha no login.
- [x] Estado global com Context API.
- [x] Persistência com `localStorage`.
- [x] Proteção de rotas para usuários logados.

---

## Desafio 04: Testes Unitários e API
Foco em testes de Controllers e Services no backend.

**Principais Desafios:**
- [x] Testes de erro para campos obrigatórios (nome e email).
- [x] Validação de chamadas de funções (`getAllUsers`).
- [x] Refatoração da rota de exclusão de usuário com testes unitários.

---

## Tecnologias
- **TypeScript**
- **React**
- **Jest** (Testes)
- **Node.js**

## Como rodar
1. Clone o repositório.
2. Instale as dependências: `npm install`
3. Execute: `npm run dev` ou `npm start`
